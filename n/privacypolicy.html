What this Bot does

Nozomi is a Discord bot that monitors Mudae bot activity inside your server. It tracks character claims, divorces, trades, gifts, roll counts, kakera values, wish-series notifications, and auction listings. It also provides roll-reset reminders, image uploading and cropping, colour generation, and fast searching of recent rolls.

Data we store

The following information is persisted in the bot's database and/or logs:

  • Discord user snowflake IDs (to associate statistics and settings with a user).
  • Discord channel and guild (server) snowflake IDs (to know where to send messages and where to listen).
  • Per-user Mudae statistics: roll counts, rolls since last wish, wish-series lists, reminder preferences, etc.
  • Per-guild configuration: tracking instance settings, enabled/disabled event types, log channel assignments, kakera-share channel and role IDs, and auction channel IDs.
  • Operational logs for debugging: usernames, guild names, member counts, commands.

Data we process but do not store

During normal operation the bot reads the following transiently and does not save it:

  • Messages sent by Mudae in tracked channels, to detect rolls, claims, and other game events.
  • Messages sent by users, to recognise bot commands and roll triggers.
  • User avatar URLs and character image URLs (used only to generate embed colours on demand).

Data sharing

No stored data is sold, shared, or disclosed to third parties. Data is used solely to operate the features of Nozomi.

Data retention and deletion

Guild data is retained as long as the bot remains in your server. If you remove the bot, you may request deletion of your server's data by contacting the developer in the support server. Individual user data (e.g. wish-series lists) can be removed on request.

Security

Data is stored in a private database on a virtual private server. Backups are maintained to guard against data loss. No passwords or payment information are ever collected.

Disclaimer

Nozomi is an unofficial addon for Mudae and is not affiliated with or endorsed by Mudae or its developer. Mudae is a separate Discord bot; please review its own privacy policy independently.
